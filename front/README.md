# FarmaEasy Frontend

Sistema de Gestao de Farmacia - Frontend

## Stack

- **React 19**
- **TypeScript 5.7**
- **Vite 6**
- **TailwindCSS 4**
- **Zustand** - State management
- **shadcn/ui** - Component library

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run tests
npm run test

# Run linting
npm run lint

# Build for production
npm run build
```
